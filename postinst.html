Posting Instructions

The purpose of this screen is to allow clubs to enter their future activities into a database. This database selects activities that are coming up in the next ten days and displays them on the "Whats Happening This Week: web page at http://graynwhite.com/whtw/whevent.htm this page is usually updated each day. In addition this database creates web pages for certain clubs that are registered with Gray and White Computing that list all of the activities for the club. Flyers or other promotional literature can be also scanned and placed on the internet with references from the event listing. Certain events require reservations in advance and arrangements can be made to display them in advance on the "Whats Happening This Week" page.

 

  1. How to enter information into the what's happening web site
  2. Sign on to the internet and access http://www.graynwhite.com/whtw/event_input.html which is pictured below.

  3. >Select the appropriate organization from the drop down box, by clicking on the thumb wheel adjacent to the word Sailing Singles (see screen below.)

    4.  

  4. Type in the proper password in the password box.
  5. Use the drop down boxes to enter the "Date From" or starting date or the date of the event if it is a single day event.
  6. If the event spans more than one day, use the drop down boxes to enter the "To Date" or ending date.
  7. Using the drop down boxes, enter the Start Time and End Time.
  8. Using the Drop down boxes enter the appropriate value in the "Day of week" field.
  9. Type in the place that the event will take place including any address or description.
  10. Type in the event name or a short description of the event.
  11. Type in the Price for members and the price for guests.
  12. You may check the boxes to indicate the nature of the event in the event type. At the present time these items are not being used and can be skipped.
  13. Click on the Submit Form button. If there are no errors Ascreen informing you that the event was accepted will appear.
  14. If another event is to be entered, use the back buttun to return to the input screen. All of the data entered is still in place so that if recurring events, such as a monthly meeting, can be entered simply by changing the date, entering the password and submitting the form.
